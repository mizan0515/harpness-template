### 버그 분류 힌트

| 분류 | 일반적 원인 | 확인 위치 |
|------|------------|----------|
| 빌드/컴파일 오류 | 누락된 import, 타입 불일치 | 빌드 출력, IDE 오류 |
| 런타임 오류 | Null 참조, 미처리 예외 | 오류 로그, 스택 트레이스 |
| 로직 오류 | 잘못된 계산, 잘못된 조건 | 서비스/비즈니스 로직 파일 |
| 상태 오류 | 상태 미동기화, 오래된 데이터 | 상태 관리, API 계층 |
| UI 오류 | 잘못된 표시, 깨진 인터랙션 | 컴포넌트, 이벤트 핸들러 |
| 데이터 오류 | 누락된 레코드, 잘못된 스키마 | DB 스키마, 시드 데이터, 로더 |
| 설정 오류 | 잘못된 환경 변수, 누락된 feature flag | 설정 파일, 배포 설정 |
| 통합 오류 | API 계약 불일치, 타이밍 문제 | API 클라이언트, 웹훅 |
